## Project Scope

This To-Do App helps users manage their daily tasks efficiently. The application allows users to add, update, delete, and mark tasks as completed. Tasks are stored locally using Room Database and the app is developed using Kotlin, Jetpack Compose, and MVVM architecture.

## Technologies Used

- Kotlin
- Jetpack Compose
- Room Database
- MVVM Architecture
- State Management

## Features

- Add Tasks
- Edit Tasks
- Delete Tasks
- Mark Tasks as Completed
- Local Data Storage with Room Database

## CRUD Operations

### Create
Users can add new tasks to the task list.

### Read
Users can view all saved tasks.

### Update
Users can edit existing tasks and update their completion status.

### Delete
Users can remove tasks from the task list.

## State Management

The application uses state management to automatically update the UI whenever task data changes, ensuring a smooth and responsive user experience.

## Conclusion

The To-Do List Mobile App provides a simple and effective way to manage daily tasks. It demonstrates the implementation of CRUD operations, Room Database integration, and state management using modern Android development practices.
